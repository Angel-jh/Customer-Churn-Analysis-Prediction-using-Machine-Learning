# Customer Churn Analysis & Prediction using Machine Learning

## Project Overview
This project focuses on predicting customer churn in the telecommunications industry using machine learning techniques. The goal is to identify customers who are likely to stop using the service based on customer behavior and service usage data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Machine Learning Models
- Logistic Regression
- Random Forest

## Project Workflow
1. Data Cleaning & Preprocessing
2. Outlier Detection
3. Feature Scaling
4. Feature Selection
5. Model Training
6. Model Evaluation
7. Data Visualization

## Results
Random Forest achieved the best model performance:
- Accuracy: 92.65%
- Precision: 82.43%
- Recall: 62.89%
- F1-Score: 71.35%

## Dataset
Telecom Customer Churn Dataset from Kaggle.
